Website Technical Info

How the site is built?

Licensing

Feel free to copy this website framework, the copyright is public domain. (excluding content)

Hosting

We have a custom build website that is hosted at gridhost. costing around R49 per month.

The sasolburgbaptist.org domain name costs around R299/per year.

Recordings

We host all our sermons, using audacity to do the recording.

This is an open source product that is free to use.

We record at 32kHz mono.

We then remove all announcements, long pauses and badly recorded testimonies from the recording.

We then normalize the audio. For normalization we use CNLevelator.

We encode to mp3 at a constant 32kbit per second which yields a file size of about 14mb per hour.

(our older recordings have higher compression, but this causes sound artefacts)

Sermon Notes

The sermon notes are generated using Microsoft Powerpoint, this then gets exported to PDF.

The PDF then gets converted to text using pdftotext command line utility.

We have a custom script to replace all illegal characters to html friendly equivalents.

The page is then hand coded to look as similar to the PDF version as possible.
